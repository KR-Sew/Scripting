There are several useful `PowerShell` scripts for managing **Printer Services** on **Windows**.  

## 📂 Managing Printer services

- 📂 Managing **Printers**
  - [`Add-PrinterPort.ps1`](./Add-PrinterPort.ps1) adding printing port to a printer
  - [`Get-NetPrintersInfo.ps1`](./Get-NetPrinersInfo.ps1) getting network printer's settings
  - [`Get-SharedPrinters.ps1`](./Get-SharedPrinters.ps1) getting shared printer's information
  - [`Get-SharedPrinters.txt`](./Get-SharedPrinters.txt) getting shared printer's info being in terminal session to use `Copy` and `Paste` and then just press `Enter`
